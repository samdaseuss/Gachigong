# Gachigong

```
“공부에 가치를 더하다”, “같이 공부하다.” 라는 중의적인 뜻을 담고있는 웹 어플리케이션으로
2023년 11월 22일 개발을 시작해 12월 4일 개발을 완료한 스프링부트 프레임워크를 사용한
팀 프로젝트입니다.
```

## 개발 환경
- JAVA 17 (openjdk 17.0.8.1)
- MySQL 8.1.0
- MacOS Ventura 13.5
- IntelliJ IDEA 2023.2.4 (Community Edition)

## 기술 스택
- Spring Boot 3.1.5
- JPA 3.1.5
- mustache 3.1.5
- Lombok 1.18.30
- Gradle 8.4
